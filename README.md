# 🌐 Personal Portfolio Website

Welcome to my personal portfolio! This project is a modern, high-performance web application designed to showcase my journey as a developer, my technical expertise, and the impactful projects I've built. It serves as a digital resume and a testament to my passion for AI, Machine Learning, and Software Development.

## 🚀 Live Demo
👉 **[View My Portfolio](http://10.29.24.73:3000/)**

---

## 📌 About the Project

This portfolio is built with a focus on **User Experience (UX)** and **Performance**. It features a sleek, dark-themed aesthetic with fluid animations and a responsive layout that works seamlessly across all devices. The goal was to create a professional yet creative space that highlights my skills in Python, React, and AI/ML while providing a clear path for potential collaborators to connect with me.

---

## 🛠️ Technologies Used

- **Frontend:** [React 19](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) (Utility-first CSS)
- **Animations:** [Motion](https://motion.dev/) (formerly Framer Motion)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Build Tool:** [Vite 6](https://vitejs.dev/)
- **Deployment:** Cloud Run (via AI Studio)

---

## ✨ Features

- **Dynamic Hero Section:** Features a typing effect and interactive background elements.
- **Glassmorphism UI:** Modern design language using frosted-glass effects and subtle gradients.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Interactive Project Showcase:** Hover effects and smooth transitions for project cards.
- **Scroll Animations:** Fade-in and slide-up animations triggered by user scrolling.
- **Contact Integration:** Direct links to LinkedIn, GitHub, and Email.

## ⚙️ Installation & Setup

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VIPULKUMAR-01/3D-Portfolio.git
   cd 3D-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:3000` to view the app.
